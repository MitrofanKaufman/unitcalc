
#### Полезные команды для проекта


- **Установка:**
  ```
  npm ci
  ```
- **Локальный запуск клиента и сервера:**
  ```
  npm run dev:client
  npm run dev:server
  ```
- **Сборка:**
  ```
  npm run build:client
  npm run build:server
  ```
- **Тесты:**
  ```
  npm run test:unit
  npm run test:int
  ```
- **Линт и формат:**
  ```
  npm run lint && npm run format
  ```
