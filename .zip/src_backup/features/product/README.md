# Product Feature

Функциональность работы с товарами Wildberries.

## Структура

- `api/` - API запросы для работы с товарами
- `components/` - Компоненты, специфичные для работы с товарами
- `hooks/` - Кастомные хуки для работы с данными товаров
- `types/` - TypeScript типы и интерфейсы

## Описание

Данный модуль отвечает за отображение и управление товарами Wildberries, включая:
- Поиск товаров
- Отображение деталей товара
- Расчет рентабельности
- Анализ цен и конкурентов
