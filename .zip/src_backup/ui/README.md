# UI Kit

Библиотека переиспользуемых компонентов интерфейса.

## Назначение

Содержит базовые UI-компоненты, которые используются по всему приложению. Эти компоненты не зависят от бизнес-логики и могут быть использованы в любом месте приложения.

## Примеры компонентов

- Кнопки (Button)
- Карточки (Card)
- Формы и поля ввода (Input, Select, Checkbox, etc.)
- Модальные окна (Modal)
- Уведомления (Toast)
- Иконки (Icons)

## Правила использования

1. Компоненты должны быть максимально переиспользуемыми
2. Не должны содержать бизнес-логику
3. Должны быть документированы с помощью Storybook
4. Должны следовать дизайн-системе приложения
