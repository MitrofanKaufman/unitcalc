# Components

Компоненты приложения, организованные по методологии Atomic Design.

## Структура

- `atoms/` - Атомарные компоненты (кнопки, инпуты, иконки)
- `molecules/` - Молекулы (комбинации атомов, например, форма поиска)
- `organisms/` - Организмы (сложные компоненты, состоящие из молекул и атомов)
- `templates/` - Шаблоны страниц (компоновка организмов в страницы)
