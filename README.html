🛍️ Калькулятор рентабельности Wildberries

MIT License TypeScript React Vite

Современный калькулятор рентабельности для маркетплейса Wildberries с современным интерфейсом и расширенной аналитикой.

🌐 Доступные версии документации

  • 📖 [README.html](./README.html) - Интерактивная HTML версия с переключением языков
  • 🇷🇺 [README.ru.md](./README.ru.md) - Полная документация на русском языке
  • 🇺🇸 [README.en.md](./README.en.md) - Full documentation in English

🚀 Возможности

  • 🔍 Поиск товаров по названию с автодополнением
  • 📊 Детальный анализ рентабельности товаров
  • 💾 Кеширование данных для ускорения работы
  • 📱 Адаптивный дизайн для всех устройств
  • 🎨 Современный UI/UX с неоморфным дизайном
  • 🌓 Поддержка тем (светлая/темная/системная)
  • 📈 Расширенная аналитика по продажам
  • 🏷 Гибкая настройка параметров расчета

🛠 Технологический стек

Фронтенд

  • Языки: TypeScript 5.x, JavaScript (ES2022+)
  • UI-фреймворк: React 18+ с хуками
  • Маршрутизация: React Router 6
  • Стилизация: TailwindCSS 3.x, PostCSS, Autoprefixer
  • Управление состоянием: React Query 4.x (удаленные данные), Zustand (клиентское состояние)

Бэкенд

  • Среда: Node.js 18+
  • Фреймворк: Express 4.x
  • Язык: TypeScript 5.x
  • База данных: MySQL 8.0+, Redis 7.x
  • ORM/ODM: TypeORM 0.3.x, Redis-om

🔧 Hooks (Хуки)

Эта папка содержит пользовательские React хуки, которые инкапсулируют сложную логику и управление состоянием для приложения.

Пользовательские хуки

useProductSearch.ts

Назначение: Пользовательский хук для управления функциональностью поиска товаров и состоянием.

Функциональность:

  • Управляет состоянием поискового запроса и результатами
  • Обрабатывает вызовы API поиска товаров с React Query
  • Предоставляет функциональность автодополнения/предложений
  • Реализует дебаунсинг поиска для производительности
  • Управляет состояниями загрузки и обработки ошибок
  • Поддерживает пагинацию и бесконечную прокрутку
  • Предоставляет навигацию с клавиатуры для предложений

Управление состоянием

  • query: Текущая поисковая строка запроса
  • results: Массив результатов поиска
  • isLoading: Индикатор состояния загрузки
  • error: Сообщение об ошибке при неудачном поиске
  • hasSearched: Булево значение, указывающее, был ли выполнен поиск

Методы

  • updateQuery(query: string): Обновляет поисковый запрос
  • performSearch(options?): Выполняет поиск с опциональными параметрами
  • clearSearch(): Очищает текущие результаты поиска
  • selectProduct(product): Выбирает товар из результатов
  • refetchSearch(): Повторно получает текущие результаты поиска

Архитектурные заметки

  • Хуки следуют лучшим практикам React для пользовательских хуков
  • Управление состоянием изолировано в области видимости хука
  • Границы ошибок и состояния загрузки обрабатываются правильно
  • Хуки предназначены для повторного использования между компонентами
  • TypeScript обеспечивает безопасность типов для всех интерфейсов хуков
  • Правильная очистка и управление памятью

Руководящие принципы использования

  • Импортируйте хуки с использованием алиаса @/hooks/
  • Всегда обрабатывайте состояния загрузки и ошибок в потребляющих компонентах
  • Используйте интерфейсы TypeScript для безопасности типов
  • Учитывайте последствия производительности использования хуков
  • Тестируйте хуки независимо при возможности

🛠 Utils (Утилиты)

Эта папка содержит утилитарные функции и вспомогательные методы, используемые во всем приложении для общих операций и преобразования данных.

Утилитарные функции

logger.ts

Назначение: Централизованная утилита логирования для отладки приложения и мониторинга.

Функциональность:

  • Логирование консоли с различными уровнями (отладка, информация, предупреждение, ошибка)
  • Структурированное логирование с метаданными
  • Управление фильтрацией и уровнями логов
  • Обработка логов в production/development
  • Отслеживание ошибок и отчетность
  • Логи мониторинга производительности

Уровни логирования

  • DEBUG: Подробная отладочная информация
  • INFO: Общие информационные сообщения
  • WARN: Предупреждающие сообщения о потенциальных проблемах
  • ERROR: Сообщения об ошибках для исключений и сбоев

Пример использования

import { logger } from '@/utils/logger';

// Различные уровни логирования
logger.debug('Отладочное сообщение', { context: 'компонент' });
logger.info('Информационное сообщение', { userId: 123 });
logger.warn('Предупреждающее сообщение', { issue: 'deprecated' });
logger.error('Сообщение об ошибке', { error, stack: true });

Архитектурные заметки

🛍️ Калькулятор рентабельности Wildberries

MIT License TypeScript React Vite

Современный калькулятор рентабельности для маркетплейса Wildberries с современным интерфейсом и расширенной аналитикой.

🚀 Возможности

  • 🔍 Поиск товаров по названию с автодополнением
  • 📊 Детальный анализ рентабельности товаров
  • 💾 Кеширование данных для ускорения работы
  • 📱 Адаптивный дизайн для всех устройств
  • 🎨 Современный UI/UX с неоморфным дизайном
  • 🌓 Поддержка тем (светлая/темная/системная)
  • 📈 Расширенная аналитика по продажам
  • 🏷 Гибкая настройка параметров расчета

🛠 Технологический стек

Фронтенд

  • Языки: TypeScript 5.x, JavaScript (ES2022+)
  • UI-фреймворк: React 18+ с хуками
  • Маршрутизация: React Router 6
  • Стилизация: TailwindCSS 3.x, PostCSS, Autoprefixer
  • Управление состоянием: React Query 4.x (удаленные данные), Zustand (клиентское состояние)

Бэкенд

  • Среда: Node.js 18+
  • Фреймворк: Express 4.x
  • Язык: TypeScript 5.x
  • База данных: MySQL 8.0+, Redis 7.x
  • ORM/ODM: TypeORM 0.3.x, Redis-om
