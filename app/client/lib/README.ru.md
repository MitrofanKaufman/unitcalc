# Библиотечные компоненты и утилиты

Эта папка содержит библиотечные компоненты и утилитарные функции, которые предоставляют общую функциональность для всего приложения.

## Описание файлов

### theme-provider.tsx
**Назначение**: React контекст-провайдер для управления темами во всем приложении.

**Функциональность**:
- Предоставляет тематический контекст всем дочерним компонентам
- Управляет состоянием темы (темная/светлая/системная)
- Обрабатывает сохранение темы в localStorage
- Предоставляет функциональность переключения темы
- Интегрируется с системными настройками темы
- Экспортирует тематические утилиты и хуки

**Ключевые особенности**:
- Реализация API контекста React
- Автоматическое сохранение темы
- Обнаружение системной темы
- Возможности переключения темы
- Поддержка TypeScript для типов темы

**Использование**:
```typescript
import { ThemeProvider, useTheme } from '@/lib/theme-provider';

function App() {
  return (
    <ThemeProvider>
      <AppContent />
    </ThemeProvider>
  );
}
```

### utils.ts
**Назначение**: Утилитарные функции и вспомогательные методы для общих операций.

**Функциональность**:
- Утилиты манипуляции строками
- Вспомогательные функции массивов и объектов
- Форматирование даты и времени
- Форматирование и преобразование чисел
- Утилиты валидации
- Вспомогательные функции оптимизации производительности

**Ключевые особенности**:
- Чистые функции без побочных эффектов
- Комплексная обработка ошибок
- Безопасность типов TypeScript
- Реализации, оптимизированные для производительности

## Подпапки

### utils/

## Архитектурные заметки

- Библиотечные компоненты предназначены для повторного использования
- Утилиты являются чистыми функциями для предсказуемого поведения
- Провайдер темы следует лучшим практикам контекста React
- Все функции тщательно протестированы
- TypeScript обеспечивает безопасность типов
- Компоненты соответствуют рекомендациям по доступности

## Зависимости

- React для контекст-провайдера
- TypeScript для определений типов
- API браузера для localStorage и системных настроек

## Руководящие принципы использования

- Импортируйте библиотечные компоненты с использованием алиаса `@/lib/`
- Используйте провайдер темы в корне дерева компонентов
- Предпочитайте утилитарные функции встроенным реализациям
- Тестируйте библиотечные функции независимо
- Документируйте сложные утилитарные функции
- Следуйте согласованным соглашениям об именовании
