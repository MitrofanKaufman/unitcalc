# Глобальные стили и CSS

Эта папка содержит глобальные CSS файлы и конфигурации стилизации, используемые во всем приложении.

## Описание файлов

### globals.css
**Назначение**: Глобальные CSS стили и базовая стилизация для всего приложения.

**Функциональность**:
- Сброс CSS и нормализация
- Глобальные переменные и пользовательские свойства CSS
- Базовые типографские стили
- Утилиты макета и системы сеток
- Базовые стили компонентов
- Значения анимации и переходов по умолчанию
- Утилиты адаптивного дизайна

**Ключевые особенности**:
- Пользовательские свойства CSS для тематизации
- Шкала адаптивной типографии
- Согласованная система отступов
- Базовые стили компонентов
- Утилиты анимации
- Поддержка темной/светлой темы

**Переменные CSS**:
- Переменные цветовой палитры
- Переменные шкалы типографии
- Переменные системы отступов
- Переменные радиуса границ
- Переменные системы теней
- Переменные шкалы z-index

### fonts.css
**Назначение**: Определения шрифтов и конфигурации типографии.

**Функциональность**:
- Объявления семейств шрифтов
- Загрузка и оптимизация шрифтов
- Определения веса и стиля шрифтов
- Конфигурации резервных шрифтов
- Конфигурации шрифтов иконок
- Стратегии загрузки веб-шрифтов

**Ключевые особенности**:
- Оптимизированная загрузка шрифтов
- Цепочки резервных шрифтов
- Интеграция шрифтов иконок
- Оптимизация производительности шрифтов
- Поддержка шрифтов между браузерами

## Архитектурные заметки

- Глобальные стили используют пользовательские свойства CSS для тематизации
- Стили организованы с использованием методологии BEM
- Типография следует согласованной шкале
- Цвета определяются с использованием семантического именования
- Адаптивный дизайн использует мобильный подход
- CSS оптимизирован для производительности

## Зависимости

- TailwindCSS для утилитных классов
- PostCSS для обработки CSS
- Библиотеки загрузки шрифтов (при наличии)
- CSS autoprefixer для совместимости браузеров

## Руководящие принципы использования

- Импортируйте глобальные стили в точке входа основного приложения
- Используйте пользовательские свойства CSS для согласованности темы
- Следуйте установленным соглашениям об именовании
- Оптимизируйте загрузку шрифтов для производительности
- Тестируйте стили на разных устройствах и браузерах
- Поддерживайте согласованные шкалы отступов и типографии
