# 🗂️ Project Change Log

This file tracks all changes, additions, deletions, and modifications made to the Unit Calculator project.

## 📋 Change Log Format

Each entry follows this format:
```
### [Date] - [Change Type]: [Description]

**Files Affected:**
- `file/path.ts` - [brief description of change]

**Details:**
[Additional context, reasoning, or technical details]

**Author:** [Name or identifier]
```

## 📊 Change Types

- **ADD**: New files or functionality added
- **DELETE**: Files or functionality removed
- **MODIFY**: Existing files or functionality changed
- **UPDATE**: Documentation, configuration, or metadata updated
- **REFACTOR**: Code restructuring without functional changes
- **FIX**: Bug fixes or corrections
