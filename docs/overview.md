# Обзор проекта

Калькулятор рентабельности Wildberries помогает продавцам принимать обоснованные решения о выводе товаров.

## 📌 Цели

- Быстрый и точный расчёт маржи
- Прогнозирование безубыточности
- Сравнительный анализ конкурентов

## 🚦 Основные функции

| Функция | Описание |
|---------|----------|
| Поиск товаров | Автодополнение, подсказки, фильтры |
| Анализ рентабельности | Учёт комиссий WB, логистики, налогов, ROI |
| Сценарии FBO/FBS | Разные модели поставки |
| История запросов | Локальный кеш + PWA offline |
| Экспорт | CSV / Excel отчёты |
