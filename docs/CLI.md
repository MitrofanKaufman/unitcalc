# Полезные команды для разработки

## Основные команды

### Запуск приложения

```bash
# Установка зависимостей
npm install

# Запуск сервера разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр собранного приложения
npm run preview
```

### Тестирование

```bash
# Запуск тестов
npm test

# Запуск тестов в режиме watch
npm test -- --watch

# Запуск тестов с покрытием
npm run test:coverage
```

### Линтинг и форматирование

```bash
# Проверка стиля кода
npm run lint

# Автоматическое исправление ошибок стиля
npm run lint:fix

# Форматирование кода
npm run format
```

### Работа с Git

```bash
# Проверка статуса репозитория
git status

# Добавление всех изменений в индекс
git add .

# Создание коммита
git commit -m "сообщение коммита"

# Отправка изменений в удаленный репозиторий
git push

# Получение изменений из удаленного репозитория
git pull
```

## Полезные команды Vite

```bash
# Анализ размера бандла
npm run build -- --report

# Запуск в режиме разработки с определенным портом
vite --port 3000
```

## Дополнительные команды

```bash
# Установка новой зависимости
npm install имя-пакета --save

# Установка dev-зависимости
npm install имя-пакета --save-dev

# Обновление всех зависимостей
npx npm-check-updates -u
npm install
```

## Отладка

```bash
# Запуск с включенным дебаггером
node --inspect-brk node_modules/.bin/vite

# Просмотр логов
npm run dev -- --debug
```

## Алиасы проекта

Проект использует алиасы для импортов. Они настроены в `vite.config.ts` и `tsconfig.json`:

- `@/*` - корневая директория `src/`
- `@/components/*` - компоненты приложения
- `@/pages/*` - страницы приложения
- `@/hooks/*` - кастомные React хуки
- `@/services/*` - сервисы для работы с API
- `@/types/*` - типы TypeScript
- `@/utils/*` - вспомогательные функции
