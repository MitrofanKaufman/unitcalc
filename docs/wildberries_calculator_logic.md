# Логика и формулы калькулятора Wildberries

## Параметры товара и комиссии Wildberries

- **Цена на WB (P)** – розничная цена продажи единицы, вводится пользователем.
- **Себестоимость (C)** – закупочная/производственная цена единицы товара.
- **Комиссия Wildberries (K\_WB)**
  ```
  K_WB = P × (p_WB / 100)
  ```

## Логистика и фулфилмент

- **Логистика от производителя (L\_prod)** – стоимость доставки единицы до склада отправителя.
- **Упаковка (U)** – стоимость упаковки товара за единицу.
- **Логистика до маркетплейса (L\_market)** – доставка до склада/ПВЗ WB.
- **Услуги фулфилмента (F)** – упаковка, маркировка и т.п.

**Общие затраты фулфилмента:**

```
Cost_fulfil = L_prod + U + L_market + F
```

## Параметры Wildberries

- **Склад отгрузки и тип поставки** – FBO или FBS, влияет на логистику и хранение.
- **Логистика до WB (L\_WB)** – доставка до ПВЗ, зависит от веса и габаритов.
- **Брак (D%)**
  ```
  Cost_brak = Q × C × (D / 100)
  ```
- **Хранение** – определяется по габаритам, складывается помесячно.
- **Выкуп (Y%)** – процент реально проданного товара.
- **Логистика с учётом выкупа (L\_WB\_adj)**
  ```
  L_WB_adj = L_WB / (Y / 100)
  ```

## Налоги и прочие расходы

- **Налоговая ставка (T%)**
  - УСН доходы:
    ```
    Tax = P × Q × (T / 100)
    ```
  - УСН доходы-расходы:
    ```
    Tax = (Revenue - Expenses) × (T / 100)
    ```
- **Прочие расходы (R)** – доп. издержки (реклама, упаковка и др.).

## Расчёты по партии

- **Инвестиции в товар (Inv\_total)** – планируемая сумма вложений.
- **Количество товара (Q)**
  ```
  Q = Inv_total / C
  ```
- **Выручка (Revenue)**
  ```
  Revenue = P × Q × (Y / 100)
  ```
- **Маржинальная прибыль (Profit)**
  ```
  Profit = Revenue - [Q × (C + L_prod + U + L_market + F + K_WB + L_WB) + Хранение + Прочие расходы + Налог]
  ```
- **Точка безубыточности (BEP)**
  ```
  BEP = Inv_total / (Profit / Q)
  ```

## Метрики на единицу товара

- **Инвестиции на 1 ед. (Invest\_1)**
  ```
  Invest_1 = C + L_prod + U + L_market + F + K_WB + L_WB_adj + (R + Tax) / Q
  ```
- **Прибыль на 1 ед. (Profit\_1)**
  ```
  Profit_1 = P × (Y / 100) - Invest_1
  ```
- **Доходность (ROI)**
  ```
  ROI = (Profit_1 / Invest_1) × 100%
  ```
- **Маржинальность (Margin%)**
  ```
  Margin% = (Profit_1 / (P × (Y / 100))) × 100%
  ```
- **Минимальная цена (P\_min)**
  ```
  P_min = Invest_1 / ((Y / 100) × (1 - p_WB / 100))
  ```
- **Максимальная скидка (MaxDiscount)**
  ```
  MaxDiscount = (P - P_min) / P × 100%
  ```

## Источники и база расчётов

Все формулы основаны на логике калькулятора [wb-calculator.ru](https://wb-calculator.ru/#calculator) и стандартных принципах юнит-экономики. Реализация производится полностью на клиенте.
