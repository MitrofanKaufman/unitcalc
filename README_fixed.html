🛍️ Калькулятор рентабельности Wildberries

MIT License TypeScript React Vite

Современный калькулятор рентабельности для маркетплейса Wildberries с современным интерфейсом и расширенной аналитикой.

🚀 Возможности

  • 🔍 Поиск товаров по названию с автодополнением
  • 📊 Детальный анализ рентабельности товаров
  • 💾 Кеширование данных для ускорения работы
  • 📱 Адаптивный дизайн для всех устройств
  • 🎨 Современный UI/UX с неоморфным дизайном
  • 🌓 Поддержка тем (светлая/темная/системная)
  • 📈 Расширенная аналитика по продажам
  • 🏷 Гибкая настройка параметров расчета

🛠 Технологический стек

Фронтенд

  • Языки: TypeScript 5.x, JavaScript (ES2022+)
  • UI-фреймворк: React 18+ с хуками
  • Маршрутизация: React Router 6
  • Стилизация: TailwindCSS 3.x, PostCSS, Autoprefixer
  • Управление состоянием: React Query 4.x (удаленные данные), Zustand (клиентское состояние)

Бэкенд

  • Среда: Node.js 18+
  • Фреймворк: Express 4.x
  • Язык: TypeScript 5.x
  • База данных: MySQL 8.0+, Redis 7.x
  • ORM/ODM: TypeORM 0.3.x, Redis-om

📄 Pages (Страницы)

Эта папка содержит компоненты страниц приложения для различных разделов калькулятора рентабельности.

Файлы в папке Pages

CalculatorForm.tsx

Назначение: Основная форма для поиска и отображения товаров с функциональностью автодополнения.

Функциональность:

  • Поиск товаров через API с автодополнением
  • Интерфейс для ввода поисковых запросов
  • Отображение результатов поиска в виде карточек товаров
  • Навигация к детальной странице товара
  • Переключение между темами (светлая/темная)
  • Адаптивный дизайн для мобильных устройств

CalculatorResultData.tsx

Назначение: Страница отображения результатов расчета рентабельности товара.

Функциональность:

  • Получение данных о товаре из API
  • Отображение рассчитанных показателей рентабельности
  • Визуализация финансовых метрик (ROI, маржа, точка безубыточности)
  • Форматирование валюты и процентов
  • Обработка ошибок загрузки данных

ProductAnalyse.tsx

Назначение: Страница детального анализа товара и его рентабельности.

Функциональность:

  • Сбор данных о товаре для анализа
  • Расчет детальной рентабельности
  • Анализ рисков и рекомендаций
  • Отображение характеристик товара
  • Интерактивные компоненты для анализа

Layout.tsx

Назначение: Основной компонент макета для всего приложения.

Функциональность:

  • Обеспечивает общую структуру приложения
  • Управляет темами (светлая/темная)
  • Интегрируется с React Router
  • Поддержка Telegram Web App
  • Адаптивный дизайн

Архитектурные особенности

  • Каждая страница следует паттернам функциональных компонентов React
  • Используется TypeScript для типизации
  • Компоненты переиспользуемы и модульные
  • Следуют принципам разделения ответственности
  • Поддержка как десктопных, так и мобильных макетов

🎨 Layouts (Макеты)

Эта папка содержит компоненты React, связанные с макетами, которые определяют общую структуру и навигацию приложения.

Компоненты макетов

Layout.tsx

Назначение: Основной компонент макета приложения, который обеспечивает общую структуру страницы.

Функциональность:

  • Обертывает все приложение с согласованной структурой макета
  • Управляет состоянием темы (темная/светлая)
  • Интегрируется с React Router для навигации
  • Предоставляет выход для вложенных маршрутов
  • Обрабатывает интеграцию с Telegram при наличии
  • Применяет классы темы к корневому элементу документа
  • Управляет состоянием макета и адаптивным поведением

Header.tsx

Назначение: Компонент заголовка, который обеспечивает навигацию и элементы брендинга.

Функциональность:

  • Отображает логотип и брендинг приложения
  • Содержит элементы навигационного меню
  • Интеграция кнопки переключения темы
  • Адаптивная навигация для мобильных устройств
  • Функциональность поиска (если применимо)
  • Доступ к учетной записи пользователя/профиля

Menu.tsx

Назначение: Компонент навигационного меню для навигации по приложению.

Функциональность:

  • Предоставляет навигационные ссылки на различные разделы
  • Обрабатывает состояние меню (открыто/закрыто)
  • Мобильная навигация
  • Интеграция с React Router
  • Подсветка активного маршрута
  • Функциональность сворачиваемого меню

Архитектурные заметки

  • Все компоненты макета следуют паттернам функциональных компонентов React
  • Компоненты предназначены для повторного использования на разных страницах
  • Интеграция темы последовательна во всех компонентах макета
  • Компоненты поддерживают как настольные, так и мобильные макеты
  • Правильная типизация TypeScript для всех пропсов и состояний

Зависимости

  • React Router для функциональности навигации
  • ThemeManager для управления состоянием темы
  • Telegram Web App SDK для интеграции с Telegram
  • Lucide React для иконок
